Public Features

  • Sort the reports table by location name, villain, and time.
  • View a report's information by clicking its view button.
  • Add a new report by clicking the add new report button in the home page or navigation bar menu.
  • When adding a report, users can select an existing location in the provided datalist. Otherwise, they could add a new location.
  • The map will display all reported nuisances with a marker

Admin Features

  • Admins will need a password to change a report's details
  • Change the status of a report by clicking the status button. The status will change from 'OPEN' to 'RESOLVED' and vice versa.
  • Delete a report by clicking the x button.
